Compilation Instructions (Java)

These instructions are intended to run on the lectura servers (as required by the assignment).
The Oracle JDBC driver is assumed to be installed at:

/usr/lib/oracle/19.8/client64/lib/ojdbc8.jar


To compile the program:

make build


To run the program:

make run USER=your_lectura_username PASS=your_lectura_password


Example:

make run USER=nate PASS=secret123


The program entry point is PetCafeApplication.
It accepts the provided credentials and connects to the Oracle database to run the menu-driven system.

Compilation Instructions (design.tex to design.pdf)

To build the design document:

make design


This will:

- run pdflatex twice inside docs/
- place all auxiliary files into docs/latex/  
- produce the final design.pdf in the project root  


To clean LaTeX auxiliary files:

make clean-docs


To remove all build output (Java + LaTeX):

make clean-all


Required Database Environment

This application requires a working Oracle account on lectura and tables loaded using:

sql/create_schema.sql
sql/insert_sample_data.sql


These scripts must be executed before running the system.
